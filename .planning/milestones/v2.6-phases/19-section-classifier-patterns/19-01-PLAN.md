substance gate (QUAL-03). Building it standalone first lets it be tested exhaustively in isolation before
any pipeline wiring exists.

    - `check_token_floor(text, metadata, *, min_tokens=DEFAULT_MIN_TOKENS) -> PredicateResult`: uses
      `constants.token_count(text)`. `count < min_tokens` -> `PredicateResult(False, f"below_token_floor:{count}<{min_tokens}")`; else `PredicateResult(True, "token_floor_ok")`.
      Cases: `check_token_floor("")` -> passed=False (0 < 3); `check_token_floor("Real content.")` (3
      tokens) -> passed=True (3 is not < 3, exact-boundary case must be tested).
    - `check_alpha_ratio(text, metadata, *, min_ratio=DEFAULT_MIN_ALPHA_RATIO) -> PredicateResult`:
      `non_ws = [c for c in text if not c.isspace()]`; if `not non_ws` -> `PredicateResult(False, "empty_text")`
      (covers empty-input edge case); else `ratio = sum(c.isalpha() for c in non_ws) / len(non_ws)`,
      `ratio < min_ratio` -> fail with reason `f"below_alpha_ratio:{ratio:.2f}<{min_ratio}"`, else pass
      with reason `"alpha_ratio_ok"`. Cases: `check_alpha_ratio("")` -> False/"empty_text";
      `check_alpha_ratio("12345")` -> ratio=0.0 -> False; `check_alpha_ratio("Real content.")` -> ratio
      ~0.92 -> True.
    - `check_link_density(text, metadata, *, max_density=DEFAULT_MAX_LINK_DENSITY, min_sample_words=_MIN_SAMPLE_WORDS) -> PredicateResult`:
      `words = text.split()`; if `len(words) < min_sample_words` -> pass with reason
      `"insufficient_sample_size"`; else `density = len(_LINK_PATTERN.findall(text)) / len(words)`,
      `density > max_density` -> fail reason `f"above_link_density:{density:.2f}>{max_density}"`, else
      pass reason `"link_density_ok"`. Cases: 3-word text -> insufficient_sample_size (True); a 10-word
      line where 5 words are markdown links `[x](y)` -> density=0.5 > 0.3 -> False; a 10-word plain
      prose line with zero links -> density=0.0 -> True.
    - `check_stopword_ratio(text, metadata, *, min_ratio=DEFAULT_MIN_STOPWORD_RATIO, min_sample_words=_MIN_SAMPLE_WORDS) -> PredicateResult`:
      `words = text.lower().split()`; if `len(words) < min_sample_words` -> pass
      `"insufficient_sample_size"`; else `ratio = sum(1 for w in words if w in STOP_WORDS_SET) / len(words)`,
      `ratio < min_ratio` -> fail `f"below_stopword_ratio:{ratio:.2f}<{min_ratio}"`, else pass
      `"stopword_ratio_ok"`. Cases: a 10-word nav-style line with zero STOP_WORDS_SET hits (e.g. "Home
      About Us Contact Sitemap Search Toggle Navigation Back Top") -> ratio=0.0 < 0.05 -> False; a
      20+-word real-prose paragraph containing "and"/"of"/"to" -> ratio clears 0.05 -> True.
    - `check_terminal_punct_ratio(text, metadata, *, min_ratio=DEFAULT_MIN_TERMINAL_PUNCT_RATIO, min_sample_words=_MIN_SAMPLE_WORDS) -> PredicateResult`:
      same `_MIN_SAMPLE_WORDS` guard; else `ratio = sum(1 for c in text if c in TERMINAL_PUNCTUATION_SET) / len(words)`,
      same fail/pass shape with reasons `f"below_terminal_punct_ratio:{ratio:.2f}<{min_ratio}"` /
      `"terminal_punct_ratio_ok"`.
    - `check_table_exemption(text, metadata) -> PredicateResult`: `metadata.get("is_table")` truthy ->
      `PredicateResult(True, "table_exempt")`; else `PredicateResult(False, "not_table")`. This is an

      tech-debt list at phase completion rather than silently treating it as already solved.
    - `check_domain_allowlist(text, metadata, *, allowlist_patterns=None) -> PredicateResult`: iterate
      `allowlist_patterns or []`; `re.search(pattern, text)` match -> immediately
      `PredicateResult(True, f"domain_allowlist_match:{pattern}")`; loop exhausted with no match (including
      the empty/None-patterns case) -> `PredicateResult(False, "no_allowlist_match")`. This is also an
      EXEMPTION predicate. Cases needed for 100% coverage: `allowlist_patterns=None` (falls through);

      with non-matching text "just prose" (loop runs, no match, falls through).
    - `compute_substance_signals(text: str) -> dict`: returns `{"token_count": token_count(text),
      "link_density": ..., "terminal_punct_ratio": ..., "stopword_ratio": ...}` — the four CLEAN-04
      annotation signals (token_count via constants.token_count; the three ratios computed directly as

      NEVER gates pass/fail — it always returns raw signal values, independent of any predicate

    - `run_predicates(text: str, metadata: dict, predicates: list[Callable]) -> PredicateResult`: iterate
      `predicates` in the given list order (never reordered/sorted — this determinism is the ordering
      edge-probe truth). For each `predicate` in order: call `result = predicate(text, metadata)`. If
      `predicate in _EXEMPTION_PREDICATES` and `result.passed` -> return `PredicateResult(True, result.reason)`
      immediately (an exemption match is an unconditional override). If `predicate not in
      _EXEMPTION_PREDICATES` and `not result.passed` -> return `PredicateResult(False, result.reason)`
      immediately (first-failing-threshold-predicate wins). Otherwise continue. If the loop completes
      without returning, `return PredicateResult(True, "all_checks_passed")`. Document explicitly in the

      Plan 19-04 is responsible for list ordering; `run_predicates` itself is order-agnostic but callers
      must place exemptions first for the override semantics to have effect before a threshold failure).
