## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| app → MinIO/S3 | Raw-zone bytes cross here; must be immutable and non-deletable |
| registry ↔ storage | Hash lookup gates whether a write happens at all |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-08 | Tampering | raw-zone object overwrite/delete | high | mitigate | Content-addressed keys + registry no-op + head_object guard + versioning/object-lock + delete-deny bucket policy (four layers, Pattern 1) |
| T-01-09 | Repudiation | dev/prod immutability divergence | high | mitigate | Never rely on S3 If-None-Match:'*' (unsupported by MinIO); enforcement is backend-portable app+bucket-policy layer |
| T-01-10 | Integrity | content hashing | medium | mitigate | hashlib.sha256 (stdlib) for raw identity — never a hand-rolled digest; xxhash reserved for non-crypto caches only |
</threat_model>
