## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-16-06 | Tampering | Domain name injection via API body | medium | mitigate | WikiExportRequest.domain has min_length=1, max_length=100 Pydantic validation; slugify in wiki.py further sanitizes before S3 key construction; existing _DOMAIN_NAME_RE pattern in api/app.py provides additional guard |
| T-16-07 | Tampering | Domain name injection via CLI | low | mitigate | Typer --domain option is a plain string; compile_wiki validates via domain_seg construction which uses only alphanumeric chars from the domain; invalid domains produce empty results (no documents found) |
| T-16-08 | Denial of Service | Large wiki compilation via API | low | accept | Wiki compilation is operator-triggered batch operation (D-12: no MCP tool); FastAPI default timeout applies; IDF threshold naturally bounds concept page count |
</threat_model>
