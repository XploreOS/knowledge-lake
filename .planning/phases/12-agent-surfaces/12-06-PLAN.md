<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| network → Streamable-HTTP /mcp | remote/browser clients may reach the local HTTP server |
| Authorization header → bearer check | untrusted credential compared against configured token |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-12-01 | Spoofing | Streamable-HTTP Host header | high | mitigate | `TransportSecuritySettings(enable_dns_rebinding_protection=True, allowed_hosts=[bind host:port, localhost:port])` rejects DNS-rebinding/foreign-Host requests |
| T-12-02 | Elevation of Privilege | write tools over network | high | mitigate | Bind 127.0.0.1 by default (no 0.0.0.0); optional bearer via KLAKE_MCP__TOKEN; KLAKE_MCP__READONLY registers only read tools |
| T-12-04 | Information Disclosure | bearer comparison | medium | mitigate | `secrets.compare_digest`, never `==`; middleware attached only when a token is configured |
| T-12-CORS | Spoofing | browser CORS | low | mitigate | `allowed_origins=[]` keeps browser origins closed (agent clients are not browsers, D-10) |
</threat_model>
