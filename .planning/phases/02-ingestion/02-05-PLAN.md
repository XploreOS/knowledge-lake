## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| hostile page → headless browser | The rendered page can attempt downloads, resource exhaustion, or navigation to internal hosts |
| adapter → followed link | Every navigated URL is attacker-influenceable |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-18 | EoP (headless browser exploitation on hostile page) | PlaywrightAdapter | high | mitigate | Downloads disabled, finite per-page navigation timeout, no arbitrary JS eval from page content, run chromium in the sandboxed container with resource caps. |
| T-02-19 | Information Disclosure (SSRF via navigated URL) | PlaywrightAdapter | high | mitigate | validate_public_url before every navigation; same-domain scope from the orchestrator. |
| T-02-20 | Repudiation / legal (robots bypass) | PlaywrightAdapter | high | mitigate | Manual robots via Protego (crawl/robots.py); disallowed → robots_blocked, no render (D-11). |
| T-02-21 | DoS (self-inflicted + oversized render) | PlaywrightAdapter | medium | mitigate | Three-tier per-host delay before navigation; rendered-content size cap (50 MB posture). |
| T-02-SC | Tampering (supply chain) | playwright==1.61.0 + chromium binary | low | accept | Approved in RESEARCH Package Legitimacy Audit (Microsoft; 5+ yrs). Browser binaries fetched at build time from official source (Pitfall 5). No [ASSUMED]/[SUS]/[SLOP]. |
