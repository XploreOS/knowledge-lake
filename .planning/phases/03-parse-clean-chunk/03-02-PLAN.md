**src/knowledge_lake/pipeline/clean.py — new pipeline stage:**

Module docstring: "Clean stage: parsed_document artifact → cleaned_document artifact. Performs boilerplate removal, whitespace normalization, language detection, and exact/near-duplicate flagging (CLEAN-01..03). Raw zone is never touched — this stage reads from and writes to the silver zone only."

Imports needed: `re`, `hashlib`, `structlog`, `typing.Optional`, `datasketch.MinHash`, `datasketch.MinHashLSH`, `lingua.Language`, `lingua.LanguageDetectorBuilder` (lazy import inside function body), all from knowledge_lake: `Settings`, `get_settings`, `StorageBackend`, `get_session`, `registry_repo`.

**Module-level constants:**

`_SILVER_PREFIX = "silver"`

`BOILERPLATE_PATTERNS: list[re.Pattern]` — a list of compiled regex patterns:
- Page headers/footers: `re.compile(r"^(?:Page \d+ of \d+|\d+)\s*$", re.MULTILINE)`
- Cookie/privacy banners: `re.compile(r"(?i)(?:this site uses cookies|accept all cookies|cookie policy)[^\n]*$", re.MULTILINE)`
- Navigation elements from HTML crawls: `re.compile(r"(?im)^(?:home|about us|contact|sitemap|skip to (?:main )?content)\s*$")`
- Repeated copyright/disclaimer lines: `re.compile(r"(?i)^(?:disclaimer|copyright \d{4})[^\n]*$", re.MULTILINE)`

Note: citations in the form "(Author, YYYY)" or "[1]" or "see §3.2" must NOT be removed. The patterns above are line-level patterns that only match entire lines of boilerplate; inline citations appear mid-sentence and will not match these patterns.

**`remove_boilerplate(text: str) -> str`:**
Apply each pattern in `BOILERPLATE_PATTERNS` via `pattern.sub("", text)`. After all substitutions, call `_normalize_whitespace(text)`. Return result.

**`_normalize_whitespace(text: str) -> str`:**
1. Replace each line: strip trailing whitespace per line using `text.rstrip()` line-by-line or a single pass regex
2. Collapse 3+ consecutive blank lines to exactly 2 blank lines: `re.sub(r"\n{3,}", "\n\n", text)`
3. Strip leading/trailing whitespace from the whole string
4. Return result

**`detect_language(text: str) -> str`:**
Lazy import `from lingua import Language, LanguageDetectorBuilder` inside the function body to avoid import-time cost.
Build a detector for ENGLISH, SPANISH, FRENCH, GERMAN, PORTUGUESE (the five most common languages in healthcare public documents):
`detector = LanguageDetectorBuilder.from_languages(Language.ENGLISH, Language.SPANISH, Language.FRENCH, Language.GERMAN, Language.PORTUGUESE).build()`
Call `detected = detector.detect_language_of(text[:2000])` (limit to first 2000 chars for speed).
If `detected` is None, return `"unknown"`.
Return `detected.iso_code_639_1.name.lower()` — this gives "en", "es", "fr", "de", "pt".
Wrap in try/except; on any ImportError log warning "lingua_not_available" and return "unknown".

**`compute_minhash(text: str, num_perm: int = 128, shingle_size: int = 5) -> MinHash`:**
Create `m = MinHash(num_perm=num_perm)`.
Tokenize text: `words = text.lower().split()`.
Generate shingles: for i in range(len(words) - shingle_size + 1): shingle = " ".join(words[i:i+shingle_size]); m.update(shingle.encode("utf-8")).
If fewer than shingle_size words exist (very short text): add the full text as a single shingle.
Return m.

**`clean(parsed_artifact_id: str, source_id: str, *, settings: Optional[Settings] = None) -> dict`:**

Docstring: "Clean a parsed_document artifact and create a cleaned_document artifact. Returns dict with artifact_id, content_hash, language, dedup_status, storage_uri."

Flow:
1. Load settings and storage: `s = settings or get_settings()`, `storage = StorageBackend(s.storage)`
2. Fetch parsed artifact from registry to get its storage_uri
3. Retrieve parsed markdown bytes from silver zone via `storage.get_object(_uri_to_key(storage_uri))`
4. Decode as UTF-8; apply `remove_boilerplate()` to get cleaned text
5. Compute SHA256 of cleaned text bytes for exact dedup: `content_hash = hashlib.sha256(cleaned_bytes).hexdigest()`
6. Check for exact duplicate: `existing = repo.get_artifact_by_hash(session, content_hash, "cleaned_document")` — if found, return existing artifact_id with dedup_status="exact_dup" (no new artifact created, same FOUND-04 pattern as parse stage)
7. Detect language: `language = detect_language(cleaned_text)`
8. Compute MinHash signature: `minhash = compute_minhash(cleaned_text, num_perm=s.clean.minhash_num_perm, shingle_size=s.clean.minhash_shingle_size)`
9. Near-dup check: build a corpus-level LSH from all existing cleaned artifacts in the registry (query `select Artifact where artifact_type='cleaned_document'`, compute their minhash from stored text). This is the transient-LSH approach — no persistent index in Phase 3 (acceptable for MVP batch sizes). For each existing cleaned artifact: retrieve its text, compute minhash, check `minhash.jaccard(existing_minhash) >= s.clean.minhash_threshold`. If any match found, set `dedup_status = "near_dup"`. Otherwise `dedup_status = "unique"`.

   Optimization note: do the near-dup scan only for the minhash query (LSH approximate lookup), not a full pairwise scan. Build an in-memory `MinHashLSH(threshold=s.clean.minhash_threshold, num_perm=s.clean.minhash_num_perm)`, insert all existing cleaned artifact minhashes (keyed by artifact_id), then query the new document's minhash. If `lsh.query(minhash)` returns any keys, set near_dup.

   For the transient LSH to work, existing cleaned artifacts' text must be retrieved. Use a batched approach: list all cleaned artifacts, retrieve their silver zone text, build the LSH, then query. This is O(n) per clean() call — acceptable for Phase 3 MVP corpus sizes (< 10,000 documents).

10. Write cleaned text to silver zone under `silver/{source_id}/cleaned/{content_hash}.md`
11. Create cleaned_document artifact in registry with metadata containing: `{"language": language, "dedup_status": dedup_status, "minhash_num_perm": s.clean.minhash_num_perm}`
12. Log completion and return: `{"artifact_id": artifact.id, "content_hash": content_hash, "language": language, "dedup_status": dedup_status, "storage_uri": cleaned_uri}`

Add `_uri_to_key(uri: str) -> str` — same implementation as in parse.py (copy verbatim; do not import from parse.py to avoid circular deps).
